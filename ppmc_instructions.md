## PPMC 

### Image Collection

1. Human user: Collect all image files in one directory, then name the directory "ppmc\_img"
2. Create a list of all of the filenames in the "ppmc\img" directory.
3. Insert these file names into the "cover\_img\_to\_title" sheet in the extraction\_data.xlsx spreadsheet file. 
4. For each image, scan the images with OCR, then enter the scanned title in the "scanned\_title" column of the "cover\_img\_to\_title" sheet.
5. Fuzzy match all of the scanned titles to find the matching title in the "book\_info" sheet of the same extraction\_data.xlsx file. Insert the titles from the "book\_info" sheet into "actual\_title" column of the "cover\_img\_to\_title" sheet. 
6. For each title with no match, re-scan the image using an LLM's native multimodal image processing to pull out the title.
7. Human user: fill in any remaining missing "actual\_title" values
8. Values for the "book\_id" column will be generated by an excel formula
9. The "new\_filename" values will be created based on the book IDs ("[book\_id]\_cover.jpg")
 
  
### Description Generation

1. For each book cover image in "ppmc\_img", run image-to-text LLM analysis to create a description.
2. Modify this description accordingly: For levels "K-R" or "K-S", the description should just be a single word or noun phrase - an object, animal, plant, person, place, or activity that features prominently in the image. For levels "1-R" or "1-S", the description should be a sentence describing what is happening or where the subject is. Examples for K-level: "dog", "teacher", "tree", "pencil"; examples for 1-level: "The dog is on the floor.", "The teacher has a book.", "The child plays the piano."
3. Insert the descriptions from step 2 into the "description" column of the "PPMC" table.

### Answer Choices

1. Double check accuracy and level-appropriateness of descriptions.
2. Add 2 distractor answers per book. The k-level ("K-R" and "K-S" in the level column) distractors should be single words or noun phrases, while the 1-level ("1-R" and "1-S" in the level column) distractors should be sentences. 
3. For each of the distractor answers, double check spelling, level appropriateness, and lack of overlapping answers. All sentences should be understandable by EFL students at a reading level equivalent to a native 1st-grade student. The distractor answers should be reasonable but clearly wrong. For 1-level distractors, the distractor answer sentences should not share subjects or predicates with the correct answer sentence. 
4. For each book, enter the correct answer and 2 distractor answers in a random order on the excel document columns "text1", "text2", and "text3"; for "answer1", enter "correct" if "text1" is the correct answer and enter "off-topic" if it is one of the distractors. Do the same for "answer2" and "text2", then "answer3" and "text3", indicating whether each answer choice is "correct" or "off-topic" (incorrect).
